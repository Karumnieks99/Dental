# Smile Studio Rīga

Dental clinic website for Smile Studio Rīga — a modern, multilingual (LV/RU/EN) landing page built with React, Vite, and Tailwind CSS.

## Live Site

[https://karumnieks99.github.io/Dental/](https://karumnieks99.github.io/Dental/)

## Features

- Multilingual support (Latvian, Russian, English)
- Before/after smile gallery with split-slider
- Services, testimonials, and contact sections
- WhatsApp quick-contact button
- Responsive design

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- GitHub Pages (via GitHub Actions)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
